== Description ==

This is a simple, scrolling banner for WordPress.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kant-banner` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
